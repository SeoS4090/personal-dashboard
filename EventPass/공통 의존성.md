# 공통 의존성 (EventPass 주변)

`EventPass.cs` 및 [[EventPassListItem]], [[PassLevelUpPopup]] 이 함께 쓰는 타입·시스템.

## 보상 표현

- **RewardItemData** — `Assets/Scripts/Investigation/InvestgationManager.cs` 에 정의 (생성자 `id, count, REWARD_TYPE, grade`).  
  [[EventPass 코어]] 의 `EventPassLevelRewardData` 가 DTO → `RewardItemData` 변환 후 [[EventPassListItem]] 에 전달.

- **RewardResultUi** — 대량 보상 시 `CommonDocManager.instance.mainHud.OpenRewardResult(...)` (`GainEffect` 분기).

## 유틸·상수

- **MKCommon** — 숫자 포맷, 토스트, `GetRewardGrade`, 패스 슬롯 JSON 등
- **MKTerm** — `GetTerm` (트랙명, `text_pass_howtoget`, `toast_pass_first`)
- **MKSpriteAtlasManager** — 트랙 아이콘 `GetUI_BG`
- **MKConstant** — `EventPass` 파일 상단 `using static MKConstant` (패키지 타입 등과 연동되는 프로젝트 전역 상수)

## UI 인프라

- **ScrollViewPro** — `Kamgam.UIToolkitScrollViewPro` (스크롤·`ScrollTo`, `OnChangePassScroll`)
- **VisualElement** 쿼리 — UXML 이름: `scroll`, `progress`, `level-list`, `track-item`, `button-levelup`, `button-how-get` 등 ([[EventPass 코어]] `Bind`)

## 콜백 타입

- **Callback**, **CallbackT&lt;long&gt;** — 프로젝트 공통 델리게이트 (`onUpdate`, `reqBuyLevelUp`)

## 기타

- **SelectValueSetting** — [[PassLevelUpPopup]] 레벨 선택 UI 템플릿
- **SeasonEventContents** — [[SeasonPass]] 베이스 클래스
