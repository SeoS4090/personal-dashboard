# 유저 데이터·네트워크 (배틀 패스 슬롯)

## MKEventPassSlotDTO

**파일:** `Assets/World/Scripts/NetworkData/MKUserDataDTO.cs`  
배틀 패스 **진행 슬롯**: `event_id`, `event_pt`, `free_level`, `premium_levels`, `pay_premiums`, `min_day`, 기간(`s_time`/`c_time`/`e_time`).  
`Data` 프로퍼티로 [[게임데이터 MKGameDataDTO]] 의 `MKEventPassBaseDTO` 지연 로드.  
`GetCurLvDt()`, `GetRecievedLevel(grade)` 등 — [[PageBattlePass]] 가 직접 사용.

## MKUserDataModel

**파일:** `Assets/World/Scripts/NetworkData/MKUserDataModel.cs`  
- `List<MKEventPassSlotDTO> battlePassList`  
- `UpdateBattlePassSlots(...)` — 패키지/미션 응답 후 갱신

## 네트워크에서 `battle_pass_list` 역직렬화

| 파일 | 용도 |
|------|------|
| `NetworkEvent.cs` | 이벤트 처리 중 `battle_pass_list` |
| `NetworkLogin.cs` | 로그인 페이로드 |
| `NetworkPackage.cs` | `MKCommon.keyRewardPassPoint` 등 보상·슬롯 갱신 |
| `NetworkMission.cs` | 미션 보상과 연동된 슬롯 업데이트 |

## 기타 참조

- `MKGameDataDTO.cs` — 일부 이벤트 콘텐츠에서 `battlePassList.Find` 패턴
- `MKCommon.cs` — 패스 슬롯 JSON 파싱 헬퍼
- [[UI 및 패키지]] — `BenefitUi`, `PromotionHub` 가 `MKEventPassSlotDTO` 조회
