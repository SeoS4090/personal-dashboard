🎮 Game 섹션 소개

게임 플레이 기록과 정보를 한 곳에서 관리하는 섹션입니다. 현재 개발 중입니다.

  • 최근 플레이한 게임 기록 및 플레이 타임 통계
  • Steam / 닌텐도 / PlayStation 계정 연동 (예정)
  • 관심 게임 위시리스트 관리
  • 게임별 메모 및 공략 노트
🔧 현재 상태
  • 섹션 UI 틀 완성 — 기능 구현 진행 중
  • Steam Web API 연동을 통한 라이브러리 불러오기 우선 개발 예정
  • API 키 설정은 ⚙️ 설정 패널에 추가될 예정입니다
🪖 Tarkov 퀘스트 라인

Escape from Tarkov 퀘스트를 트레이더별로 연결선 그래프로 시각화하는 기능입니다.

  • 1차: 수동 JSON 데이터로 퀘스트 노드/선 표시
  • 2차: 필터(트레이더, 레벨, 맵) + 검색
  • 3차: 진행 상태 저장(localStorage) 및 진척도 시각화
🧱 데이터 소스 메모
  • Fandom 페이지 HTML은 구조 변경 가능성이 있어 파싱 내구성이 낮습니다.
  • 초기 버전은 프로젝트 내부 JSON 스키마를 우선 고정하는 방식을 권장합니다.
  • 추후 자동 수집이 필요하면 별도 스크립트로 배치 갱신하는 흐름이 안전합니다.
