# 기획 볼트 (Obsidian)

이 폴더를 Obsidian **볼트 루트**로 연다. (저장소 루트 `LogDec01` 가 아닌 `기획` 폴더를 연다.)

## 첫 실행

1. Obsidian → Open folder as vault → 이 `기획` 폴더 선택
2. [[Index]] 노트 열기 (`00_Index/Index.md`)
3. **할 일만 볼 때**: [[할일-인덱스]] (`TODO list/할일-인덱스.md`)
4. 플러그인 권장: **Linter**, **Templater** (`_templates/` 경로 지정)

## 폴더

| 경로 | 용도 |
|------|------|
| `00_Index/` | 최상위 MOC |
| `MOC/` | 도메인별 Map of Content |
| `notes/{system,content,data,worldview}/` | 원자 노트 (유형당 1단계만) |
| `_templates/` | Templater 템플릿 |
| `_meta/` | 규격·가이드 (일반 기획 노트 아님) |
| `TODO list/` | 실행 체크리스트 ([[할일-인덱스]]) |

저장소 전체 규칙: 상위 `../AGENTS.md`, `../README.md`
