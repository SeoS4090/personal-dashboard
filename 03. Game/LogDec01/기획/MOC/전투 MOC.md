# 전투 MOC

## 개요

전투는 **완전 자동**이다. **할 일**: [[전투-할일]]. [[자동-전투-시스템]] 이 슬롯 순환·쿨다운·타겟 AI로 카드를 발동하고, [[HP-방어-시스템]] 과 [[대미지 계산 공식]] 이 피해를 처리한다. 상태 이상은 개념당 원자 노트로 분리하며, [[덱-구성-규칙]] 의 슬롯·덱이 전투 전 설계 수단이다.

## 핵심 축

- 규칙·공식: `notes/system/`
- 상태·스킬 단위: 개념당 한 노트
- 데이터: `notes/data/` — 스키마만, 장문 기획은 system 노트로
