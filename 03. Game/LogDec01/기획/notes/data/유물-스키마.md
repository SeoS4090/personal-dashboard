# 유물-스키마

## 1. 용도

- [[상점-시스템]]·[[보상-시스템]]·직업 시작 유물이 참조하는 패시브 아이템 형식.

## 2. 스키마

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | string | ✅ | relic_rusty_shield |
| name | string | ✅ | 표시명 |
| description | string | ✅ | 효과 설명 |
| rarity | enum | ✅ | common, uncommon, rare, boss |
| hooks | string[] | | battle_start, on_crit, round_start… |
| params | object | | 수치 파라미터 |

## 3. 제약·검증

- 런당 동일 id 유물 중복 불가
- 시작 유물은 `rarity: common` 고정

## 4. 예시 row

```json
{
  "id": "relic_rusty_shield",
  "name": "낡은 방패",
  "description": "전투 시작 시 보호막 5",
  "rarity": "common",
  "hooks": ["battle_start"],
  "params": { "block": 5 }
}
```
