## 3. effects 표기

| effects (요약) | 의미 |
|----------------|------|
| `block:N` | 보호막 N |
| `heal:N` | HP N 회복 |
| `poison:N` / `bleed:N` / `burn:N` | 상태 스택 |
| `stun:1` | [[상태이상-기절]] 1라운드 |
| `slow:1` | [[상태이상-둔화]] 1중첩 |
| `weak:1` / `empower:1` | 약화/강화 1라운드 |
| `mark:1` | 표적 표식 (피해 ×1.2) |
| `taunt:2` | 도발 2라운드 |
| `cleanse:1` | 디버프 1 제거 |
| `evade:20` | 회피 +20% 1라운드 |
| `shield_slam` | 보호막×0.5 + power direct |
