# 카드-스키마

## 1. 용도

- [[자동-전투-시스템]]·[[상점-시스템]]·[[직업-시작-덱-표]] 가 참조하는 카드 레코드 형식.

## 2. 스키마

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | string | ✅ | 고유 id (예: w_strike) |
| name | string | ✅ | 표시명 |
| classTags | string[] | | warrior, mage, common… |
| cost | int | ✅ | 쿨다운 라운드 1~4 |
| power | int | | direct 위력 |
| scale | float | | 공격력 계수 (기본 1.0) |
| target | enum | ✅ | nearest, lowest_hp, random, all_enemies, self |
| tags | string[] | | physical, magic, dot, heal… |
| rarity | enum | ✅ | common, uncommon, rare |
| effects | object[] | | 상태 부여 등 |

## 3. 제약·검증

- `cost` 1~4
- `target=self` 이면 power는 힐·보호막·버프만

## 4. 예시 row

```json
{
  "id": "w_strike",
  "name": "강타",
  "classTags": ["warrior"],
  "cost": 1,
  "power": 6,
  "scale": 1.0,
  "target": "nearest",
  "tags": ["physical"],
  "rarity": "common",
  "effects": []
}
```

## 5. 변경·마이그레이션

- 업그레이드 카드: id 접미사 `_plus` (예: w_strike_plus)
