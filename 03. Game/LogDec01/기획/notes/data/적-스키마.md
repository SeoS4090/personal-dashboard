# 적-스키마

## 1. 용도

- [[적-풀-Act1]] 등 전투 스폰 테이블이 참조하는 적 유닛 형식.

## 2. 스키마

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | string | ✅ | e_goblin_melee |
| name | string | ✅ | 표시명 |
| act | int | ✅ | 1~3 |
| tier | enum | normal, elite, boss | ✅ |
| hp | int | ✅ | MaxHP |
| atk | int | ✅ | 기본 공격력 |
| def | int | | def% 0~50 |
| speed | int | | 기본 10 |
| tags | string[] | melee, ranged, magic, undead |
| pattern | string | | AI 패턴 id |
| goldDrop | int[] | | min,max 전투 보상 (런 골드는 [[보상-Act-표]]) |
| relicDrop | float | | 엘리트/보스 유물 확률 |

## 3. 제약·검증

- normal: 1~2체 스폰 / elite: 1체 / boss: 1체
- Act N 적은 Act N·N+1 노드에 등장 가능

## 4. 예시 row

```json
{
  "id": "e_goblin_melee",
  "name": "고블린 도끼병",
  "act": 1,
  "tier": "normal",
  "hp": 28,
  "atk": 6,
  "def": 0,
  "speed": 10,
  "tags": ["melee", "goblin"],
  "pattern": "melee_basic"
}
```
