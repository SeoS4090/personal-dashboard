# 직업-전사

## 1. 개요

- 역할: 탱커·근접. [[HP-방어-시스템]] 의 보호막·높은 MaxHP(80) 중심.
- 런 시작 시 [[직업-시작-덱-표]] 의 전사 덱·시작 유물 적용.

## 2. 구성·흐름

| 항목 | 내용 |
|------|------|
| 패시브 | 받는 피해 **10%** 를 [[HP-방어-시스템]] 보호막으로 전환(HP 감소는 90%만) |
| 시작 유물 | `relic_rusty_shield` **낡은 방패**: 전투 시작 보호막 5 |
| 전용 태그 | `taunt`, `shield_bash`, `warrior` |
| 타겟 AI | [[자동-전투-시스템]] `nearest` 고정, 도발 카드 시 적 타겟 자신 |

### 대표 카드 방향 (초안 id)

- `w_strike` — 근접 direct
- `w_block` — 보호막 8
- `w_taunt` — 2라운드 도발
- `w_shield_slam` — 보호막×0.5 + direct

## 3. 연출·UI

- 팔레트: 강철 회색 + 붉은 망토 악센트
- 도트: 방패·검, 32×48 스프라이트

## 4. 보상·밸런스

- 상점·보상 풀: `warrior` 태그 카드 가중
- [[대미지 계산 공식]] 직접 딜은 낮고 생존으로 장기전
