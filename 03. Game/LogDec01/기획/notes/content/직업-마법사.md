# 직업-마법사

## 1. 개요

- 역할: AoE·원소. MaxHP 60, 낮은 [[HP-방어-시스템]] def.
- [[상태이상-연소]]·빙결(후속) 중심 빌드.

## 2. 구성·흐름

| 항목 | 내용 |
|------|------|
| 패시브 | 원소 상태([[상태이상-연소]] 등) 부여 시 **stacks +1** 추가 |
| 시작 유물 | `relic_cracked_focus` **균열 수정**: 전투 첫 `magic` 카드 쿨 -1 |
| 전용 태그 | `fire`, `frost`, `mage`, `aoe` |

### 대표 카드 방향

- `m_bolt` — 단일 magic
- `m_fireball` — `all_enemies` + 연소 2
- `m_frost_nova` — 전체 + 둔화 1
- `m_channel` — self 강화 1스택

## 3. 연출·UI

- 팔레트: 보라·청록, 주문 이펙트 8×8 파티클

## 4. 보상·밸런스

- Act 후반 AoE scaling 주의 — [[보상-시스템]] 희귀 카드로 조절
