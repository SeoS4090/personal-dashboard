# 직업-궁수

## 1. 개요

- 역할: 단일 딜·치명. [[대미지 계산 공식]] 크리·표적 표식 활용.

## 2. 구성·흐름

| 항목 | 내용 |
|------|------|
| 패시브 | 치명타 시 **다음 카드 쿨 -1** ([[자동-전투-시스템]]) |
| 시작 유물 | `relic_old_bow` **낡은 활**: 치명 +5% |
| 전용 태그 | `mark`, `rapid`, `archer` |
| 타겟 AI | `lowest_hp` 가중 +20% |

### 대표 카드 방향

- `a_shot` — direct, `lowest_hp`
- `a_mark` — 표적 표식(statusMul ×1.2)
- `a_double` — 2히트 낮은 위력
- `a_pierce` — 보호막 무시 50% (관통 초안)

## 3. 연출·UI

- 팔레트: 녹색·갈색, 화살 궤적 1px

## 4. 보상·밸런스

- 크리 상한 50%와 유물 중첩 설계
