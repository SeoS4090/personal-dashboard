# 직업-도적

## 1. 개요

- 역할: DoT·회피. [[상태이상-독]]·[[상태이상-출혈]] 과 [[대미지 계산 공식]] +15% 시너지.

## 2. 구성·흐름

| 항목 | 내용 |
|------|------|
| 패시브 | 독·출혈 보유 적에게 direct **+15%** |
| 시작 유물 | `relic_poison_vial` **독침**: 전투 시작 무작위 적 독 2 |
| 전용 태그 | `bleed`, `poison`, `evade`, `rogue` |
| 회피 | 기본 5%, 카드로 상승 |

### 대표 카드 방향

- `r_stab` — direct + 출혈 2
- `r_poison` — 독 3
- `r_evade` — self 회피 +20% 1라운드
- `r_finisher` — 출혈 소모 burst

- 팔레트: 보라·검정, 단검 잔상

## 4. 보상·밸런스
