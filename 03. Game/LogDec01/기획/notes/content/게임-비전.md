# 게임-비전

## 1. 개요

**한 줄**: 판타지 던전을 한 직업의 덱으로 자동 전투하며, 매 전투 뒤 상점에서 덱과 유물을 키우는 로그라이크.

플레이어는 [[직업 MOC]] 에서 직업을 고르고, [[게임-코어-루프]] 로 런을 진행한다. 전투는 [[자동-전투-시스템]] 이 전부 처리하므로, 재미의 중심은 **덱·슬롯 설계**와 **상점 의사결정**에 있다.

## 2. 핵심 판매점

- 덱 빌딩 성장감 + 자동 전투의 관전·최적화 재미
- 5직업 고유 특성으로 매 런 빌드 차별화
- 전투 승리 후 **필수 [[상점-시스템]]** 으로 페이싱·선택 집중

## 3. 비목표 (1차 스코프)

- 실시간 조작, 멀티, PvP, 5인 파티
- 전투 중 플레이어 입력(카드 사용·타겟 지정)

## 4. 연출·UI — 도트 아트 가이드
