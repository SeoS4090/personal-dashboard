# 상태이상-둔화

## 1. 기획 의도 및 핵심 루프

- [[자동-전투-시스템]] 의 speed를 낮춰 행동 빈도 감소.

## 2. 핵심 규칙 (Core Rules)

| 항목 | 값 |
|------|-----|
| 효과 | `speed × 0.7` (30% 감소) |
| 중첩 | 최대 2회 (0.7² ≈ 49% 속도) |
| 지속 | 2 라운드, 갱신 시 지속 리셋 |

- `floor(speed/10)` 행동 횟수에 반영

## 3. 예외 처리 및 엣지 케이스

- 최소 행동 1회/라운드 유지

## 4. 데이터 구조 및 인터페이스

- Status: `{ id: "slow", stacks: 1|2, duration }`
