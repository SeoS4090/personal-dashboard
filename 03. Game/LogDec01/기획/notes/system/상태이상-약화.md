# 상태이상-약화

## 1. 기획 의도 및 핵심 루프

- 주는 피해 감소 디버프. [[대미지 계산 공식]] 의 `statusMul`에 반영.

## 2. 핵심 규칙 (Core Rules)

| 항목 | 값 |
|------|-----|
| 효과 | 주는 **direct** 피해 × 0.75 |
| 중첩 | 없음, 갱신만 |
| 지속 | 2 라운드 |
| DoT | 약화는 direct에만 적용 (독·출혈·연소 미적용, 초안) |

- [[상태이상-강화]] 와 동시 시: 곱연산 아닌 **강화 우선** 후 약화 (net × 1.0) — 상쇄

## 3. 예외 처리 및 엣지 케이스

- 버프 스킬의 힐·보호막에는 미적용
