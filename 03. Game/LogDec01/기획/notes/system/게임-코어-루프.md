# 게임-코어-루프

## 1. 기획 의도 및 핵심 루프

- 한 런 = [[직업 MOC]] 에서 **직업 1명** 선택 → 맵 노드 선택 → [[자동-전투-시스템]] 전투 → 승리 시 **필수** [[상점-시스템]] → 덱 편집(선택) → 반복 → 막 보스 → 승리 또는 HP 0 패배
- 플레이어 개입: 직업 선택, 맵 루트, 상점·덱 편집. 전투 중 입력 없음.

## 2. 핵심 규칙 (Core Rules)

### 런 구조

| 항목 | 값 (초안) |
|------|-----------|
| 막(Act) | 3 |
| 층 수 | 막당 8~10 노드 + 막 보스 1 |
| 승리 조건 | Act3 보스 처치 |
| 패배 조건 | 전투 중 HP 0 ([[HP-방어-시스템]]) |

### 노드 유형

| 노드 | 효과 |
|------|------|
| 일반 전투 | 적 1~2체, 승리 → 상점 |
| 엘리트 | 강화 적, 승리 → 상점 + [[보상-시스템]] 엘리트 보상 |
| 상점(고정) | 맵에 드문 상점 노드(전투 없이 구매만) |
| 이벤트 | 텍스트 선택·[[재화-골드]]/HP 변동 |
| 휴식 | MaxHP의 30% 회복 (상한 현재 MaxHP) |
| 보스 | 막 종료 전투, 승리 → 상점 + 보스 보상 |

### 상점 진입

- 일반·엘리트·보스 **승리 직후 [[상점-시스템]] 자동 진입, 스킵 불가**
- 구매 없이 «나가기»만 가능

### 재화 (런 내)

- [[재화-골드]]: 상점·덱 서비스
- 영혼 조각: 메타 진행용 placeholder (1차 미구현)

## 3. 예외 처리 및 엣지 케이스

- 보스 전 노드 수 미달 시: 맵 생성기가 보스 직전 휴식 1칸 보장 (구현 TODO)
