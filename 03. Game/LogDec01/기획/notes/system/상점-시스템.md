# 상점-시스템

## 1. 기획 의도 및 핵심 루프

- [[게임-코어-루프]] 에 따라 일반·엘리트·보스 **승리 직후 자동 진입**, 스킵 불가.
- 플레이어는 [[재화-골드]] 로 카드·유물·덱 서비스를 사거나 «나가기»만 한다.

## 2. 핵심 규칙 (Core Rules)

### 진입

- 트리거: 전투 승리 직후
- 맵 «상점 노드»: 전투 없이 동일 UI

### 6칸 구성 (랜덤, 방문마다 재생성)

| 칸 | 내용 | Act1 가격 (G) |
|----|------|----------------|
| 2 | 카드 (직업+공용, [[카드-스키마]] rarity 가중) | 50~120 |
| 1 | 유물 ([[유물-스키마]]) | 150~250 |
| 1 | 소모품 (1회성, 전투 시작 자동 사용 옵션) | 40~80 |
| 1 | 덱 서비스: 제거 75 / 업그레이드 100 | 고정 |
| 1 | 스탯 부적: MaxHP+8(80), 공격+2(70) 등 | 70~80 |

- 가격: [[재화-골드]] Act 스케일 적용

### 리롤

- 50G, **방문당 1회** (유물로 추가 리롤 가능, 후속)

### 카드 풀 가중

| rarity | 상점 출현 |
|--------|-----------|
| common | 60% |
| uncommon | 30% |
| rare | 10% |

- 현재 직업 `classTags` + `common` 태그 카드

## 3. 예외 처리 및 엣지 케이스

- 덱 제거 시 선택 UI로 카드 1장 지정
- 유물 중복 id 구매 불가 → 칸 비활성 또는 교체 풀

## 4. 데이터 구조 및 인터페이스

- ShopVisit: `{ slots: ShopSlot[], rerollUsed: bool }`
- ShopSlot: `{ type, itemId, price }`
