# 상태이상-기절

## 1. 기획 의도 및 핵심 루프

- 행동 불가 구간으로 전투 템포 조절. [[자동-전투-시스템]] 에서 카드 발동·일반 공격 스킵.

## 2. 핵심 규칙 (Core Rules)

| 항목 | 값 |
|------|-----|
| 효과 | N **라운드** 동안 행동 불가 |
| 기본 지속 | 1 라운드 (카드·스킬별 `duration` 오버라이드) |
| 중첩 | 지속 시간만 갱신(스택 수 없음), 최대 3 라운드 |
| 크리티컬 | 기절 중 [[대미지 계산 공식]] 에서 **치명 불가** (`isCrit = false`) |

### 부여·저항

- 부여 시: `저항 = target.stunResist` (기본 0, 보스 +50%)
- 성공: `rand() > 저항` 이면 `duration` 적용

### 해제

- 지속 라운드 종료 시 자동 해제
- 피해 누적 ≥ MaxHP×15% 단일 히트 시 50% 확률 조기 해제 (초안)
- [[직업-사제]] 정화: 디버프 1개 제거 시 기절 포함 가능

## 3. 예외 처리 및 엣지 케이스

- 보스: Act1 보스 기절 면역, Act2+ 저항 50%만 (draft)
- 기절 중 보호막·DoT는 정상 처리

## 4. 데이터 구조 및 인터페이스

- Status: `{ id: "stun", duration, resistRoll }`
