# 재화-골드

## 1. 기획 의도 및 핵심 루프

- 런 내 유일 주요 화폐. [[보상-시스템]] 으로 유입, [[상점-시스템]]·[[덱-구성-규칙]] 제거/업그레이드로 소비.

## 2. 핵심 규칙 (Core Rules)

| 항목 | 규칙 |
|------|------|
| 보유 | 런당 정수, 상한 없음 |
| 런 종료 | 전부 소멸 (메타 [[게임-코어-루프]] 영혼 조각은 별도) |

### 가격 스케일 (Act)

```
표시가격 = 기준가 × (1 + 0.15 × (Act - 1))
```

- Act1 기준가는 [[상점-시스템]] 표 참고

### 주요 소비

| 용도 | Act1 기준 (G) |
|------|----------------|
| 카드 구매 | 50~120 |
| 유물 | 150~250 |
| 소모품 | 40~80 |
| 덱 제거 | 75 |
| 덱 업그레이드 | 100 |
| 스탯 부적 | 70~80 |
| 상점 리롤 | 50 (방문당 1회) |

## 3. 예외 처리 및 엣지 케이스

- 이벤트로 음수 골드 불가 (최소 0)
