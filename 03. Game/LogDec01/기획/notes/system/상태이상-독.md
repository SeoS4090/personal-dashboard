# 상태이상-독

## 1. 기획 의도 및 핵심 루프

- 지속 압박 DoT. [[직업-도적]] 빌드·[[직업-마법사]] 일부 카드와 연동.

## 2. 핵심 규칙 (Core Rules)

| 항목 | 값 |
|------|-----|
| 중첩 | `stacks` 정수, 상한 99 |
| 틱 | **라운드 종료** 시 `stacks` 만큼 HP 피해 (direct 파이프라인) |
| 보호막/방어 | [[HP-방어-시스템]]·[[대미지 계산 공식]] DoT 규칙 적용 |

- 부여 시 기존 stacks에 `+부여량`
- [[직업-도적]]: 독 보유 적에게 direct +15%

## 3. 예외 처리 및 엣지 케이스

- 정화 1회에 독 전부 제거 또는 stacks 절반 — 카드별 명시 (기본: 전부)
