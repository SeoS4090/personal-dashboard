# 상태이상-출혈

## 1. 기획 의도 및 핵심 루프

- «맞을수록 아픈» DoT. [[직업-도적]] 핵심 메카닉.

## 2. 핵심 규칙 (Core Rules)

| 항목 | 값 |
|------|-----|
| 중첩 | `stacks`, 상한 99 |
| 틱 | 대상이 **direct 피해를 받을 때마다** 추가로 `stacks` 피해 |
| 보호막/방어 | 적용 |

- 라운드 종료 시 stacks **유지**(자연 감소 없음, 초안)
- 일부 카드 «출혈 소모»: stacks 0으로 만들고 즉시 `stacks × 2` burst

## 3. 예외 처리 및 엣지 케이스

- DoT끼리 출혈 트리거 안 함
- [[직업-도적]] +15% direct는 출혈 보유 시에만

## 4. 데이터 구조 및 인터페이스
