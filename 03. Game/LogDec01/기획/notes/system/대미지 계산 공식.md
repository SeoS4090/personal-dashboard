# 대미지 계산 공식

## 1. 기획 의도 및 핵심 루프

- [[자동-전투-시스템]] 의 모든 공격·스킬이 이 축을 따른다.
- 예측 가능한 수치 + [[상태이상-강화]] 등으로 빌드 차별화.

## 2. 핵심 규칙 (Core Rules)

### 즉시 피해 (direct)

```
raw = (공격력 + 카드위력) × 스킬계수
critMul = isCrit ? 1.5 : 1.0   // [[상태이상-기절]] 중 크리 불가 → critMul = 1.0, isCrit 강제 false
afterDef = raw × critMul × (1 - 방어력%)
statusMul = Π(상태 배율)       // 약화 0.75, 강화 1.25, 표적 표식 +1.2 등
final = max(1, floor(afterDef × statusMul))
```

이후 [[HP-방어-시스템]] 순서: 회피 → 보호막 → HP.

### 치명타

- 기본 5% + 스탯/유물, **상한 50%**
- [[직업-궁수]]: 치명 시 다음 카드 쿨 -1 ([[자동-전투-시스템]])

### 태그

| 태그 | 비고 |
|------|------|
| `physical` | def 적용 (1차 단일 def) |
| `magic` | def 적용 (후순위: 마법 def) |
| `dot` | DoT 파이프라인 |

### DoT

| 상태 | 계산 | 보호막 |
|------|------|--------|
| [[상태이상-독]] | 라운드 종료 `스택` 피해 | 적용 |
| [[상태이상-출혈]] | 피격 시 `스택` 추가 피해 | 적용 |
| [[상태이상-연소]] | 라운드 시작 `스택` 피해 | **무시** |

- [[직업-도적]]: 출혈/독 대상에게 direct 피해 +15%

### 직업·상태 연동 예

- 표적 표식(궁수): `statusMul` × 1.2
- [[상태이상-약화]]: 주는 피해 × 0.75
- [[상태이상-강화]]: 주는 피해 × 1.25

## 3. 예외 처리 및 엣지 케이스

- final 최소 1 (0 방지)
- 면역: 보스 일부 «기절 면역» 등은 상태 노트에 정의

## 4. 데이터 구조 및 인터페이스

- DamageEvent: `{ source, target, base, tags[], isCrit, final }`
- 카드 `power`, `scale` 필드: [[카드-스키마]]
