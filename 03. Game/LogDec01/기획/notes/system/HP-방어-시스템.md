# HP-방어-시스템

## 1. 기획 의도 및 핵심 루프

- Slay the Spire **Block(보호막)** + RPG **HP** 혼합. [[대미지 계산 공식]] 과 연동한다.
- HP 0 시 전투 패배 → [[게임-코어-루프]] 런 종료.

## 2. 핵심 규칙 (Core Rules)

| 개념 | 규칙 |
|------|------|
| **HP** | 현재/최대. 0 → 패배. 휴식 노드·일부 카드·[[직업-사제]] 힐로 회복 |
| **보호막** | 피해 흡수 우선. **라운드 종료 시 전부 소멸** |
| **방어력(def)** | 피해 `× (1 - def%)`, 1차 단일 수치 (물리/마법 분리는 후순위) |
| **회피** | 확률로 해당 공격 무효 ([[직업-도적]]·카드) |

### 피해 적용 순서

`회피 판정` → `보호막` 감소 → `방어력 %` → `HP`

- DoT([[상태이상-독]], [[상태이상-출혈]]): 보호막·방어력 적용 후 HP
- [[상태이상-연소]]: 라운드 시작 피해는 **보호막 무시**, 방어력은 적용

### 전사 패시브 연동

- [[직업-전사]]: 받는 피해 10%를 보호막으로 전환(해당 피해만큼 HP 감소는 90%)
- 라운드 종료 보호막 소멸 규칙은 동일(잔류 보호막 유물은 별도)

### 시작 스탯 (초안)

| 직업 | MaxHP | def% |
|------|-------|------|
| 전사 | 80 | 10 |
| 마법사 | 60 | 0 |
| 궁수 | 70 | 5 |
| 도적 | 65 | 5 |
| 사제 | 70 | 0 |

## 3. 예외 처리 및 엣지 케이스

- 힐 초과분: [[직업-사제]] 패시브 시 50% 보호막
- 보호막 음수 불가

## 4. 데이터 구조 및 인터페이스

- Unit: `{ hp, maxHp, block, def, evasion }`
