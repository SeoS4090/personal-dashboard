# LogDec01 기획 인덱스

이 문서는 **기획 볼트의 유일한 진입점**이다. 폴더를 깊게 탐색하지 말고, 아래 MOC와 메타 가이드를 따라간다.

## 이 볼트의 구조

- **원자 노트**: `notes/` — 개념 하나당 한 파일
- **도메인 MOC**: `MOC/` — 주제별 거미줄 인덱스
- **규격·가이드**: `_meta/` — 작성 규칙 (일반 노트와 구분)
- **템플릿**: `_templates/` — 새 노트 생성용
- **할 일 목록**: `TODO list/` — 실행만 모은 체크리스트 ([[할일-인덱스]])

## 빠른 작업 흐름

1. 작업 주제에 맞는 **도메인 MOC** 를 연다.
2. MOC에 없는 개념이면 `notes/` 에 원자 노트를 만들고 MOC에 한 줄 요약과 링크를 추가한다.
3. `status`·`updated`·`related` 를 갱신한다.
