📅 Google Calendar 연동

Google Calendar API를 연결하면 대시보드에서 일정을 바로 확인할 수 있습니다.

  1. Google Cloud Console 접속 후 새 프로젝트 생성
  2. API 및 서비스 → 라이브러리에서 Google Calendar API 검색 후 활성화
  3. 자격 증명 → API 키 만들기로 API 키 생성 후 복사
  4. 대시보드 ⚙️ 설정 → Google Calendar에 API Key 입력
  5. Calendar ID: 본인 Gmail 주소 또는 primary 입력 후 저장
🔒 API 키 보안 설정
  • Cloud Console → 자격 증명 → API 키 편집에서 HTTP 참조자 제한 추가 권장
  • 허용 URL에 localhost/* 또는 실제 배포 도메인 입력
  • API 키는 브라우저 로컬 스토리지에만 저장되며 외부 서버로 전송되지 않습니다
📰 NewsAPI.org 연동

NewsAPI.org를 통해 키워드 기반 뉴스를 실시간으로 가져옵니다.

  1. newsapi.org 회원가입 (무료 플랜 제공)
  2. 로그인 후 My Account 페이지에서 API Key 복사
  3. 대시보드 ⚙️ 설정 → News API에 API Key 입력
  4. 관심 키워드를 쉼표로 구분해 입력 (예: 게임, AI, 개발, 스타트업)
  5. 저장 후 뉴스 패널로 이동하면 자동으로 로드됩니다
⚠️ 무료 플랜 제한사항
  • 무료 플랜은 하루 100회 API 호출 제한 (일반 사용에 충분)
  • 기사 본문은 일부만 제공 — 전문은 카드 클릭 후 원본 링크에서 확인
  • 해외 뉴스 중심이므로 영문 키워드를 함께 추가하면 더 많은 결과를 얻을 수 있습니다
  • 한국 뉴스 위주로 받으려면 키워드에 korea를 포함하거나 한글 키워드 사용
📝 메모 사용법

별도 API 없이 브라우저 로컬 스토리지에 저장하는 빠른 메모 기능입니다.

  • + 새 메모 버튼 클릭 → 제목과 내용 입력 후 저장
  • 메모 카드를 클릭하면 편집 모드로 전환됩니다
  • 카드 우상단 ✕ 버튼으로 개별 삭제 가능
  • 메모 목록은 최근 수정 순으로 정렬됩니다
💾 저장 방식 안내
  • 인터넷 연결 없이도 작동하며 서버에 데이터를 전송하지 않습니다
  • 브라우저 데이터(캐시·쿠키) 초기화 시 메모가 삭제될 수 있으니 중요한 내용은 별도 백업 권장
  • 같은 기기의 같은 브라우저에서만 메모를 확인할 수 있습니다

🎮 Game 섹션 소개

게임 플레이 기록과 정보를 한 곳에서 관리하는 섹션입니다. 현재 개발 중입니다.

  • 관심 게임 위시리스트 관리
  • 게임별 메모 및 공략 노트
🔧 현재 상태
  • 섹션 UI 틀 완성 — 기능 구현 진행 중
  • Steam Web API 연동을 통한 라이브러리 불러오기 우선 개발 예정
  • API 키 설정은 ⚙️ 설정 패널에 추가될 예정입니다

⚡ Dev 섹션 소개

개인 개발 프로젝트와 GitHub 활동을 대시보드에서 관리하는 섹션입니다.

  • GitHub 저장소 목록 및 최근 커밋 히스토리
  • 이슈 · PR 상태 트래킹
  • 개인 프로젝트 메모 및 TODO 관리
  • 기술 스택 통계 시각화
🔧 현재 상태
  • 섹션 UI 틀 완성 — 기능 구현 진행 중
  • GitHub API (Personal Access Token) 연동 우선 개발 예정
  • PAT(Personal Access Token)은 GitHub Settings → Developer settings에서 발급
  • API 키 설정은 ⚙️ 설정 패널에 추가될 예정입니다
