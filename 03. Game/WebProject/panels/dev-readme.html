⚡ Dev 섹션 소개

개인 개발 프로젝트와 GitHub 활동을 대시보드에서 관리하는 섹션입니다.

  • GitHub 저장소 목록 및 최근 커밋 히스토리
  • 이슈 · PR 상태 트래킹
  • 개인 프로젝트 메모 및 TODO 관리
  • 기술 스택 통계 시각화
🔧 현재 상태
  • 섹션 UI 틀 완성 — 기능 구현 진행 중
  • GitHub API (Personal Access Token) 연동 우선 개발 예정
  • PAT(Personal Access Token)은 GitHub Settings → Developer settings에서 발급
  • API 키 설정은 ⚙️ 설정 패널에 추가될 예정입니다
