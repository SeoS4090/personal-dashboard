📅 Google Calendar 연동 (OAuth 2.0)

OAuth 2.0으로 Google 계정에 직접 로그인하여 비공개 캘린더를 포함한 모든 캘린더를 불러옵니다.

  1. Google Cloud Console 접속 후 새 프로젝트 생성
  2. API 및 서비스 → 라이브러리에서 Google Calendar API 검색 후 활성화
  3. 자격 증명 → OAuth 클라이언트 ID 만들기 선택 → 유형: 웹 애플리케이션
  4. 승인된 JavaScript 원본에 서비스 중인 주소 추가 (예: https://username.github.io)
  5. 생성된 클라이언트 ID 복사 후 ⚙️ 설정 → Google Calendar → OAuth 2.0 Client ID 에 입력 후 저장
  6. Google 계정 연결 버튼 클릭 → 팝업에서 계정 선택 및 권한 허용
  7. 연결 후 설정 → 캘린더 목록에서 원하는 캘린더 ID를 추가
📋 자주 사용하는 캘린더 ID
  • 내 기본 캘린더: primary
  • 대한민국 공휴일: ko.south_korea#holiday@group.v.calendar.google.com
  • 회사/공유 캘린더: 캘린더 설정 → 캘린더 통합 → 캘린더 ID 확인
🗓️ 뷰 전환 및 탐색
  • 월별: 달력 그리드로 한 달 일정 한눈에 확인
  • 주별: 7일 시간대별 상세 뷰 (오전 7시 ~ 오후 10시)
  • 일정: 앞으로 30일간 일정 목록 (캘린더 색상 구분)
  • ‹ › 버튼으로 이전/다음 기간으로 이동
  • 주 시작 요일은 ⚙️ 설정에서 일요일 / 월요일 선택 가능
📰 NewsAPI.org 연동

NewsAPI.org를 통해 키워드 기반 뉴스를 실시간으로 가져옵니다.

  1. newsapi.org 회원가입 (무료 플랜 제공)
  2. 로그인 후 My Account 페이지에서 API Key 복사
  3. 대시보드 ⚙️ 설정 → News API에 API Key 입력
  4. 관심 키워드를 쉼표로 구분해 입력 (예: 게임, AI, 개발, 스타트업)
  5. 저장 후 뉴스 패널로 이동하면 자동으로 로드됩니다
⚠️ 무료 플랜 제한사항
  • 무료 플랜은 하루 100회 API 호출 제한 (일반 사용에 충분)
  • 기사 본문은 일부만 제공 — 전문은 카드 클릭 후 원본 링크에서 확인
  • 해외 뉴스 중심이므로 영문 키워드를 함께 추가하면 더 많은 결과를 얻을 수 있습니다
  • 한국 뉴스 위주로 받으려면 키워드에 korea를 포함하거나 한글 키워드 사용
📝 메모 사용법

별도 API 없이 브라우저 로컬 스토리지에 저장하는 빠른 메모 기능입니다.

  • + 새 메모 버튼 클릭 → 제목과 내용 입력 후 저장
  • 메모 카드를 클릭하면 편집 모드로 전환됩니다
  • 카드 우상단 ✕ 버튼으로 개별 삭제 가능
  • 메모 목록은 최근 수정 순으로 정렬됩니다
💾 저장 방식 안내
  • 인터넷 연결 없이도 작동하며 서버에 데이터를 전송하지 않습니다
  • 브라우저 데이터(캐시·쿠키) 초기화 시 메모가 삭제될 수 있으니 중요한 내용은 별도 백업 권장
  • 같은 기기의 같은 브라우저에서만 메모를 확인할 수 있습니다
