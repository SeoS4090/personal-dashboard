## 4. 데이터 필드 해석 참고

- **sq**: 스쿼드별 현재 상태  
  - `n`: 현재 병력, `i`: 초기 병력, `j`: 누적 피해(또는 부상), `w`: 전사 등.
- **t (msg 내)**  
  - 1: 일반 공격, 3: 스킬/특수 효과, 4: 군단 참전, 5/6: 스킬, 8/9: 추가 스킬/효과 등.
- **m**: 이동/공격 이벤트 (a=공격자 sq, d=방어자 sq, t=타입, p=파라미터).
- **b**: 버프/디버프 (i=효과 id, v=값, t=대상/지속 등).
- **leader**: 현재 공격/방어 리더 함대 키.
- **sa / sqs**: 스쿼드 활성 여부 및 소속 스쿼드 ID 목록 (비우면 전멸·제거).
