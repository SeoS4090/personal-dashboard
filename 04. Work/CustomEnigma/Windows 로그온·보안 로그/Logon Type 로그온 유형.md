# Logon Type (로그온 유형)

이벤트 **4624**, **4634** 등에 포함되는 **LogonType**은 “어떤 종류의 로그온/세션”인지를 나타내는 숫자입니다. WorkTimer·출근 감지에서는 **대화형**과 **잠금 해제**에 해당하는 유형을 골라 씁니다.

## WorkTimer에서 사용하는 유형

| 값 | 이름(통칭) | 출근/해제 맥락 |
|----|------------|----------------|
| **2** | Interactive | 콘솔에 **대화형으로 로그온**한 세션 |
| **7** | Unlock | **워크스테이션 잠금 해제**(이전 세션 유지) |

→ [[GetWorkTime 필터와 이벤트 읽기 이슈]]에서 실제 필터 조건과 연결됩니다.

## 자주 같이 보는 다른 유형 (참고)

| 값 | 통칭 | 비고 |
|----|------|------|
| **3** | Network | 네트워크 로그온(SMB 등) |
| **4** | Batch | 작업 스케줄러 등 |
| **5** | Service | 서비스가 로컬 시스템 등으로 로그온 |
| **10** | RemoteInteractive | RDP 등 |

> [!tip] Win+L 잠금
> 잠금 자체는 **4800**, 해제는 **4801** 이벤트로 보는 편이 직관적입니다(감사 정책 필요). 로그온 이벤트만 볼 때는 **4624 + LogonType 7**이 “해제 직후 로그온” 신호로 쓰입니다. → [[잠금·해제 이벤트 4800·4801]]
